Модуль ngx_http_empty_gif_module

Пример конфигурации
Директивы
     empty_gif

Модуль ngx_http_empty_gif_module выдаёт однопиксельный прозрачный GIF.

Пример конфигурации

location = /_.gif {
    empty_gif;
}

Директивы

Синтаксис: empty_gif;
Умолчание: —
Контекст: location

Разрешает в содержащем location выдавать однопиксельный прозрачный GIF.
